INTRODUCE

	Check_websites is a check tools that automatic test the website availability.
	When the website is unavailable,it's send warning email to admin.

=====================================================================================

INSTALLATION

	Decompression check_website.tar.gz,then you can use it!

=====================================================================================

USAGE:
	check_website  have two configure files:admin.py WebSites.conf
	As your required to config admin.py and WebSites.conf.
	Finally,execute:sh check.sh
